Math Worksheet Platform - Integration Guide

Embed interactive worksheets anywhere with minimal setup. No registration required, instant scoring, optional progress tracking.

Method 1: Direct Link Integration

The simplest way to integrate. Perfect for LMS systems, classroom websites, or any platform that supports links.

<!-- Simple link integration --> <a href="https://yourapp.com/worksheet/EX02" target="_blank"> Take Algebra Quiz - Expressions & Equations </a> <!-- Or as a button --> <button onclick="window.open('https://yourapp.com/worksheet/EX02', '_blank')"> Start Worksheet </button>

Method 2: iframe Embedding

Embed worksheets directly in your page. Good for seamless user experience within your platform.

<!-- Basic iframe embedding --> <iframe src="https://yourapp.com/worksheet/EX02" width="100%" height="600" frameborder="0" allowfullscreen> </iframe> <!-- Responsive iframe --> <div style="position: relative; width: 100%; height: 0; padding-bottom: 75%;"> <iframe src="https://yourapp.com/worksheet/EX02" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0"> </iframe> </div>

Method 3: API Integration

Full control over the user experience. Fetch worksheet data and handle scoring in your own interface.

// Fetch worksheet data const response = await fetch('https://api.yourapp.com/worksheet/EX02', { headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' } }); const worksheet = await response.json(); /* Response: { "id": "EX02", "title": "Expressions & Equations", "description": "Translate word problems into mathematical expressions", "questions": [ { "id": "q1", "expression": "3X - 2", "description": "Two less than three times a number x", "when_x": 3, "type": "number" } // ... more questions ] } */ // Submit answers for scoring const scoreResponse = await fetch('https://api.yourapp.com/worksheet/EX02/submit', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ session_id: 'user-session-123', student_id: 'optional-student-id', answers: { "q1": 7, "q2": 5, "q3": 8, "q4": 1, "q5": 6 } }) }); const score = await scoreResponse.json(); /* Response: { "session_id": "user-session-123", "score": { "correct": 5, "total": 5, "percentage": 100 }, "feedback": [ {"question": "q1", "correct": true, "user_answer": 7, "correct_answer": 7}, // ... more feedback ], "time_spent": 245 } */

📊 API Endpoints

  • GET /api/worksheet/{id} - Get worksheet questions
  • POST /api/worksheet/{id}/submit - Submit answers for scoring
  • GET /api/student/{id}/progress - Get student progress
  • POST /api/webhook/score - Webhook for score notifications
